# HTML and CSS Requirements

-Semantic tags must be used: <header>, <nav>, <main>, <section>, <article>, <footer>, <search>

-Use grid layout or flexbox for page layout

Use media queries to adjust for mobile devices and tablets

Verify that the HTML is valid using the W3C validator

# Directory Structure

The project is organized into folders and files as follows:

images/

Contains the images used in the project.

styles/

Contains the CSS files that define the project's style.

index.html

Main project file that structures the content and links the styles
