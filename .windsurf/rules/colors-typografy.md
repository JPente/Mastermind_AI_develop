#color palette:

-F5EFE7: Main background. Fondo general cálido, beige suave.
-EDE6DE: Secondary background. Secciones, tarjetas y bloques sobre fondo claro.
-2F2F2F: Header/Nav background. Barra superior fija, contraste elegante.
-3A3A3A: Navigation menu background (hover). Ligero contraste, mantiene sobriedad.
-4A4A4A: Navigation menu normal text. Links del menú en estado normal.
-6A6A6A: Navigation menu hover/active text. Estado activo y hover en navegación.
-1E1E1E: Body text on light background. Texto principal sobre fondos claros.
-F5EFE7: Text on dark background. Texto sobre header u otras zonas oscuras.
-2B4B60: Links. Color para enlaces dentro de texto (suave, profesional).
-406782: Link hover. Efecto hover para enlaces.
-6D90A6: Visited links. Ligeramente más tenue para diferenciar recorrido.
-2F2F2F: Main button background. Botón primario (oscuro y protagonista).
-FFFFFF: Main button text. Texto del botón principal.
-3A3A3A: Main button hover. Un tono más claro al pasar el mouse.
-FFFFFF: Secondary button background. Botón secundario (contraste invertido).
-2F2F2F: Secondary button border & text.
-2F2F2F: Footer background. Pie de página en gris/negro suave.
-E8DCCF: Footer text. Texto y enlaces en el footer.
-C8B49A: Footer links. Color suave y elegante para enlaces de pie de página.
-B59E83: Footer hover. Hover más cálido y marcado.

Cuerpo de texto / párrafos / enlaces: Nunito Sans con peso 400 ο 500.
